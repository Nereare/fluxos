# Fluxos

Repositório de fluxos operacionais e POPs relativos a um serviço específico.

## Contribuir

Fluxos precisam ser discutidos com coordenação, gerência e diretoria, portanto todos os _pull requests_ não previamente discutidos e autorizados serão recusados.
